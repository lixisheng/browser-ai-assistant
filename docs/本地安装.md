# 本地安装未打包插件

## 安装步骤

1. 在项目根目录执行构建命令：

```powershell
npm run build:extension
```

2. 打开 Chrome 或 Edge 的扩展管理页面。
3. 打开“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择项目下的 `dist` 目录：

```text
F:\ProgramFiles\Code\chrome-extension\browser-ai-assistant\dist
```

## 常见错误

### 清单文件缺失或不可读取

如果选择的是项目根目录：

```text
F:\ProgramFiles\Code\chrome-extension\browser-ai-assistant
```

浏览器会提示“清单文件缺失或不可读取”，因为根目录没有 `manifest.json`。本项目使用 Vite 构建，浏览器扩展清单会在构建后输出到 `dist/manifest.json`，因此必须加载 `dist` 目录。

## 验证

构建后确认以下文件存在：

```text
dist/manifest.json
dist/index.html
dist/background/index.js
dist/content/index.js
```
