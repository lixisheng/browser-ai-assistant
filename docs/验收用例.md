# Browser AI Assistant MVP 验收用例

## 1. 侧边栏入口

- 点击插件图标后，侧边栏应打开。
- 使用快捷键 `Ctrl+Shift+Y` 后，侧边栏应打开。
- 在页面右键菜单点击“打开 AI 助手”后，侧边栏应打开。

## 2. 首次使用

- 未配置任何模型时，对话输入区域显示“请先配置 API Key 后再开始对话”。
- 未配置任何模型时，“发送”按钮不可点击。

## 3. 模型配置与请求

- 用户可以配置多个渠道、多个模型。
- 用户可以在对话输入框处切换当前模型。
- 用户可以切换流式响应模式。
- OpenAI Chat Completions 请求包含 `model`、`messages`、`temperature`、`max_tokens`、`stream`。
- Anthropic Messages 请求将系统提示词放在顶层 `system` 字段。
- API Key 校验失败时展示失败原因，但仍允许保存配置。
- 模型请求失败时展示“请求失败，请重试”，失败消息不保存为正式聊天记录。

## 4. 网页内容提取

- URL 正则未匹配时，回退到 `<html>` 下全局可见文本。
- CSS/XPath 未命中、执行失败或结果为空时，回退到全局可见文本。
- 一条规则中的多行 CSS/XPath 按行顺序提取并拼接。
- 超长网页从开头截取，并提示用户“内容被截断，请细化 CSS/XPath”。

## 5. 聊天记录

- 聊天记录仅保存大模型对话数据字段白名单。
- 不保存 URL、原始 DOM、网页标题等额外网页元数据。
- 默认按会话产生顺序排序。
- 用户可以手动修改标题。

## 6. 标题生成

- 配置标题总结模型时，基于对话数据生成标题。
- 未配置标题总结模型时，使用 `网站 title + yyyy-MM-dd HH:mm:ss`。
- 标题生成失败时，使用默认标题并允许用户修改。

## 7. Chrome Sync 加密备份

- API Key、链接凭据、聊天记录和配置同步前必须加密。
- 对称加密密钥不参与同步。
- 配置界面明确提示“忘记密钥将无法恢复已加密的同步数据”。
- 一个备份前缀只保留一个远程备份。
- 手动恢复直接覆盖本地，不做合并。
- Chrome Sync 配额超限时，展示“备份失败：同步数据超过 Chrome Sync 配额，请减少历史记录或关闭部分同步内容”。
